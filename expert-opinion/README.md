# Expert Opinion Outputs

Use this folder for consultant-style answers produced by ISO/BPMN custom agents.

## Path Convention

For each topic, create a folder and write one file per consultant:

- `expert-opinion/<topic>/topic-iso15489.md`
- `expert-opinion/<topic>/topic-iso9001.md`
- `expert-opinion/<topic>/topic-iso27001.md`
- `expert-opinion/<topic>/topic-iso30301.md`
- `expert-opinion/<topic>/topic-iso22301.md`
- `expert-opinion/<topic>/topic-iso19510-bpmn20.md`

## Topic Naming

Use lowercase kebab-case topic names, for example:

- `expert-opinion/document-retention-policy/`
- `expert-opinion/vendor-onboarding-workflow/`
- `expert-opinion/incident-response-governance/`
